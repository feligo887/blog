## 用原生Node创建自己的脚手架工具

### 1. 创建自定义的全局指令
- 新建一个'bin'文件夹
- 在 bin文件夹中新建一个cli.js文件
- 在项目根目录打开命令行，输入：'npm init' or 'npm init -y'（package.json会自动引入bin目录下的文件）
- 在项目根目录打开命令行，输入：'npm link' 把 package.json中bin 配置命令挂载到全局。
- 在cli.js写入 '#! /usr/bin/env node'
- - #! 告诉系统这个是一段运行脚本
- - /usr/bin/env 需要的环境路径
- - node 需要的环境

### 目录结构
```
- bin
  |-- cli.js
- lib
  |-- core
    |--- action.js
    |--- conmmand.js
    |--- dowload.js
    |--- frawork.js
- config.js
- package.json
```
